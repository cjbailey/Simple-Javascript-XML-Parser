Simple-Javascript-XML-Parser

Created with CodeSandbox

A really basic, light-weight Javascript XML parser using a good dollop of Regex. Not intended to be a comprehensive, robust XML parser that deals with all eventualities but should handle simple XML okay.

View on Github

How to use

let xml = "<root lang='en'><foo a='qwe'><foobar>hello world</foobar></foo></root>";
let xmlDoc = new XmlDocument(xml);
xmlDoc.toString();

XmlElement

xmlDoc.children ... (An array of XmlElement objects) ...

XmlAttribute

let element = xmlDoc.children[0];
element.attributes ... (An array of XmlAttribute objects) ...

XmlTextContent

let text = new XmlTextContent("hello world");
let element = new XmlElement("p", ...attributes);
element.append(text);
